Description

In this project, I work with David Simonetti to build a Transformer Architecture Detoxifier (as well as a Toxifier), in order to observe the potential of machine learning based censoring over traditional word bank censoring. We use a parallel toxic to detoxified dataset in order to train a machine translation task, measuring reductions in toxicity and preservation of sentence meaning compared to a word bank baseline. This project was completed as an assignment for a Fall 2023 Natural Language Processing course.
